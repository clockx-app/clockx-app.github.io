Troubleshooting

Installer blocked by SmartScreen

Click “More info” → “Run anyway” if you trust the official source. Ensure you downloaded from the link above.

Clock not visible on desktop

Check if the app is running in the tray. Toggle always‑on‑top and transparency in settings; move the clock to a visible monitor.

Alarms don’t ring

Verify volume and output device, check Do Not Disturb/Focus Assist, and ensure the app is allowed to play sounds.

High CPU usage

Try a simpler skin, disable animations, and restart the app. If it persists, reinstall using the official installer.

Start with Windows

Enable “Start with Windows” in settings. If blocked by policy, ask your system administrator.

Wrong time zone

Update Windows time zone settings. For extra clocks, use the World Clock feature and set the desired cities.

Uninstall Guide

  1. Open Apps & Features

    Press Win+I → Apps → Installed apps (or classic Control Panel → Programs and Features).

  2. Find ClockX

    Scroll the list or search for “ClockX”.

  3. Uninstall

    Click Uninstall and confirm. Follow on‑screen steps until completion.

  4. (Optional) Remove settings

    Delete the ClockX settings folder from your user profile if you want a fully clean slate next time.
